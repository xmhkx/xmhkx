### 🛠️ Featured Projects

*   **Stacker Arcade Game:** Developed a Stacker-style game using a PIC16F1829 and ATmega328P. I wrote the firmware for UART communication, I2C LCD control, and integrated an 8x32 WS2812B LED matrix, earning 2nd place in the Microcomputer Systems 1 project.
*   **AI Lane-Following Robot:** Trained a YOLOv7-tiny model using 1,500 images for lane detection and implemented multiple C++ PID control algorithms to improve motor response. I designed the control system by integrating ToF, ultrasonic, and IMU sensors for autonomous navigation.
*   **EPICS Robotic Arm:** Programmed an Arduino-based robot and custom arm using C/C++. This project won the UTSA EPICS competition out of 60+ students.
